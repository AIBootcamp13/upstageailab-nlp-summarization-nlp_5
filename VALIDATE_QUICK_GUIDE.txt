실험 환경 검증 스크립트 빠른 가이드
=====================================

파일 위치:
- Python 검증 스크립트: scripts/validation/validate_experiment_env.py
- 사전 테스트 스크립트: scripts/validation/prerun_test.py  
- 통합 검증 스크립트: scripts/validation/validate_before_run.sh
- 제출 검증 스크립트: scripts/validation/validate_submission.py
- 상세 가이드 문서: docs/06_troubleshooting/validation/VALIDATION_README.md

빠른 실행:
1. 통합 검증 (권장):
   bash scripts/validation/validate_before_run.sh

2. Python 상세 검증:
   python scripts/validation/validate_experiment_env.py

3. 빠른 기능 테스트:
   python scripts/validation/prerun_test.py --quick

4. 제출 파일 검증:
   python scripts/validation/validate_submission.py

문제 발생 시 docs/06_troubleshooting/validation/VALIDATION_README.md 참조
