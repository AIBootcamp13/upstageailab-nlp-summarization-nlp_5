# Shell Script 오류 패턴

## 발생 파일
- run_main_7_experiments.sh
- 기타 실행 스크립트

## 반복 횟수
3회 이상

## 주요 증상
- printf 문법 오류
- conda 환경 활성화 실패
- 변수 참조 오류
- 경로 문제

## 실패한 수정 시도들
- printf 형식만 수정 → 다른 문법 오류
- conda 경로 하드코딩 → 환경별 차이 무시
- 부분적 수정 → 스크립트 실행 흐름 파괴

## 주의사항
- shellcheck로 검증 필수
- 환경별 차이 고려
- 변수 스코프 확인
- 실행 권한 확인
