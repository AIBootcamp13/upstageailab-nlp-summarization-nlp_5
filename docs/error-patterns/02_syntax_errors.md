# SyntaxError: unterminated triple-quoted string literal

## 발생 파일
- data_utils.py (주로)

## 반복 횟수
15회 이상 (10시간 동안)

## 주요 증상
- 여러 줄 문자열(""" 또는 ''') 닫기 실패
- 문자열 내부의 따옴표 처리 오류
- 주석과 문자열 혼동

## 실패한 수정 시도들
- 따옴표만 추가 → 다른 위치에서 문제 발생
- 문자열을 잘못된 위치에서 닫음
- 이스케이프 처리 실패

## 주의사항
- 문자열 시작과 끝을 명확히 확인
- 다중 줄 문자열 사용 시 들여쓰기 주의
- 파일 전체에서 따옴표 매칭 검증
