## 📋 포함 문서

### 📈 [데이터셋 분석](./dataset_analysis.md)
- DialogSum 데이터셋 구조 및 특성 분석
- 데이터 품질 평가 및 노이즈 패턴 식별
- 대화 길이 분포 및 주제별 특성

### 🔧 [텍스트 전처리](./text_preprocessing.md)
- 한국어 대화 데이터 정제 방법

2. **탐색적 데이터 분석** - 통계적 특성 및 패턴 분석  
3. **전처리 전략 수립** - 최적의 정제 방법론 결정
4. **전처리 적용** - 실제 데이터 정제 및 검증
